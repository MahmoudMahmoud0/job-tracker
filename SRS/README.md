# Job Tracker Docs

This repository contains the documentation site for the Job Tracker platform. It is organized as an SRS and implementation guide using MkDocs Material.

## Local development

1. Create a Python virtual environment.
2. Install dependencies:

```bash
pip install mkdocs-material
```

3. Start the documentation server:

```bash
mkdocs serve
```

4. Open the local URL shown in the terminal.

## Documentation scope

The site covers:

- product vision and business context
- functional and non-functional requirements
- domain model and data design
- UI and workflow specifications
- REST API contracts
- deployment and operations
- testing strategy
- delivery roadmap
