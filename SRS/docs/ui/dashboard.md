# Dashboard

## Objective

The dashboard provides an immediate view of the user's job-search health and next actions.

## Key Widgets

- total applications
- applications by status
- interviews in the next 7 days
- overdue follow-ups
- weekly application trend
- top opportunity sources

## UX Notes

- The first screen should emphasize actionability over dense reporting.
- Empty states should encourage adding the first company or application.
