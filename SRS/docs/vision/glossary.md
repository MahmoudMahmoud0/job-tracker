# Glossary

| Term | Definition |
| --- | --- |
| Application | A user's submission for a role at a company |
| Pipeline Status | The current lifecycle state of an application |
| Follow-up | A reminder or action linked to an application or interview |
| Interview | A scheduled or completed screening, technical round, or final round |
| Contact | A person related to a company or application |
| Source | Where the opportunity was discovered, such as LinkedIn or referral |
| Offer | A formal employment proposal associated with an application |
