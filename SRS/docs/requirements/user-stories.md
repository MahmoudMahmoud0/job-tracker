# User Stories

## Application Tracking

- As a job seeker, I want to log a new application quickly so that I can keep my pipeline current.
- As a job seeker, I want to change the status of an application so that I can reflect progress over time.
- As a job seeker, I want to view all applications by stage so that I can understand where my search stands.

## Interview Workflow

- As a user, I want to schedule interviews with reminders so that I do not miss key events.
- As a user, I want to record interview notes per round so that I can prepare for future conversations.

## Analytics

- As a user, I want a dashboard showing totals and conversion rates so that I can improve my strategy.
- As a coach, I want to review application trends so that I can advise clients with evidence.
